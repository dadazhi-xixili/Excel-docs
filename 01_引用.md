# 01 引用

## 01.01 单元格引用

| 行\列 | A   | B   | C   | ... |
| ----- | --- | --- | --- | --- |
| 1     | "a" | "b" | "c" | ... |
| 2     | "d" | "e" | "f" | ... |
| 3     | "g" | "h" | "i" | ... |
| ...   | ... | ... | ... | ... |

---

**Excel 中，单元格上方的字母（A、B、C...）代表列，单元格左侧的数字（1、2、3...）代表行。**

| 引用方法 | 描述                     | 示例                        | 结果                                  |
| :------- | :----------------------- | :-------------------------- | :------------------------------------ |
| 直接引用 |                          | A1                          | "a"                                   |
| 区域引用 |                          | A1:B2                       | {"a","b";"d","e"}                     |
| 区域引用 | 根据引用范围内的极点引用 | B1:B3:A2:C2(可以视作 A1:C3) | {"a","b","c";"d","e","f";"g","h","i"} |
| 行引用   | 对整行引用               | 1:1                         | {"a","b","c"...}                      |
| 列引用   | 对整列引用               | A:A                         | {"a";"d";"g"...}                      |
| 交集引用 | 取两个区域的交集         | B1:B3 A2:C2                 | "e"                                   |
