# 02&ensp;数据类型

## 02.01&ensp;值类型

| 类型   | 描述                                           | 示例                                                                          |
| ------ | ---------------------------------------------- | ----------------------------------------------------------------------------- |
| 数值   | 包含整数、浮点数、正负数、科学计数数、**日期** | `1`、`1.0`、`-1.0`、`1e-10`、`2024/01/01`                                     |
| 字符串 | 由字母、符号、汉字等组成的字符序列             | `"hello"`、`"hello world"`、`"123"`                                           |
| 布尔值 | `TRUE`、`FALSE`                                | `TRUE`、`FALSE`                                                               |
| 错误值 | 运算中产生的错误值                             | `#N/A`、`#DIV/0!`、`#VALUE!`、`#REF!`、`#NAME?`、`#NULL!`、`#NUM!`、`#ERROR!` |
